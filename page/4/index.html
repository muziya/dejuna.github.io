解决 误按 command ＋ q

习惯使用快捷键切换，效率。
但最近经常使用 Chrome 浏览器切换标签页的时候，比如经常切换到的第一个标签页 command ＋ 1. 但是稍不注意，或者说小手够不着，哈哈，会没按到1，却按倒了 q , 但是大家都知道，command + q 是系统默认强制退出的快捷键，所以会把你所有打开的标签页全部关闭，比如你看东西看了一半，或者邮件写了一半，或者更衰一点，你用隐身模式开启了很多标签页，那你瞬间就想哭了。
