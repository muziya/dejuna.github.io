解决 误按 command ＋ q

发表于 2015-10-03 | 分类于 小技 |

习惯使用快捷键切换，效率。
但最近经常使用 Chrome 浏览器切换标签页的时候，比如经常切换到的第一个标签页 command ＋ 1. 但是稍不注意，或者说小手够不着，哈哈，会没按到1，却按倒了 q , 但是大家都知道，command + q 是系统默认强制退出的快捷键，所以会把你所有打开的标签页全部关闭，比如你看东西看了一半，或者邮件写了一半，或者更衰一点，你用隐身模式开启了很多标签页，那你瞬间就想哭了。

解决办法

解决方法肯定是设置改掉这个快捷键，换成一个怎么也不会产生误按的一个快捷键组合。
但是当你打开 Chrome 的设置选项，发现没有这个设置项。
所以就看看 Mac 系统设置吧，找到 键盘 － 快捷键 － 应用快捷键。
此时你会发现右边栏下面，有 + - ，点击 ＋ ，应用程序选择 Chrome ，菜单名称 就是你要设置的快捷键所要产生的动作，我们就到 Chrome 看一下，可以看到退出到完整名称 —— 退出Google Chrome，所以填上这个，然后在下一栏，按下一个比较不容易误按到快捷键方式。
此时你再到 Chrome 浏览器，你会看到退出的快捷键方式以及变了。再也不用担心手误产生不必要的麻烦了。
